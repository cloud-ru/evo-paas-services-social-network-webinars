# Нагрузочное тестирование (k6)

Данная директория содержит набор скриптов для нагрузочного тестирования бэкенда социальной сети с использованием [k6](https://k6.io/).

## Предварительные требования

Для запуска тестов необходимо установить k6:

- **macOS**: `brew install k6`
- **Windows**: `choco install k6`
- **Linux**: Инструкции на [официальном сайте](https://k6.io/docs/getting-started/installation/).

## Установка

Установите зависимости для поддержки типов (необязательно, для корректной работы автодополнения в IDE):

```bash
cd loadtest
npm install
```

## Конфигурация

Основные настройки находятся в файле `social-network.config.js`:

- `BASE_URL`: URL бэкенда (по умолчанию `http://localhost:3000`).
- `options`: Параметры сценария (ramping VUs, thresholds). Здесь настроены этапы наращивания нагрузки (ramp-up), пиковая нагрузка и условия прохождения теста (лимиты по времени ответа и проценту ошибок).

## Запуск тестов

Все команды запускаются из директории `loadtest`.

### Стандартный тест

Запуск сценария с параметрами, определенными в `social-network.config.js`:

```bash
npm test
```

### Короткий тест

Запуск короткого теста для проверки работоспособности:

```bash
npm run test:small
```

## Структура проекта

- `scripts/social-network-journey.js`: Основной сценарий, имитирующий пользовательский путь.
- `lib/api-client.js`: Типизированный клиент для взаимодействия с API социальной сети.
- `social-network.config.js`: Глобальная конфигурация тестов и сценариев k6.

## Сценарий тестирования

Скрипт `social-network-journey.js` имитирует реалистичное поведение пользователя:

1. **Регистрация и Вход**: Каждый цикл начинается с создания уникального пользователя и получения токена доступа.
2. **Случайная Активность**: Пользователь выполняет от 3 до 8 случайных действий:
   - Просмотр ленты новостей (50%)
   - Создание постов или лайки (25%)
   - Отправка личных сообщений (15%)
   - Поиск пользователей (10%)
3. **Задержки**: Между действиями добавлены паузы ("think time"), а также задержки после операций записи, чтобы симулировать реальное взаимодействие.
